GET Tenant by ID

GET /api/tenants/:id
Example: GET /api/tenants/4
Response: 200 OK
{
  "id": 4,
  "name": "JONA PALACIO",
  "room_number": "2",
  "contact": "09560376568",
  "monthly_rent": "1500.00",
  "status": "Active"
}

POST Create Tenant

POST /api/tenants
Headers: Content-Type: application/json
Body:
{
  "name": "Juan Dela Cruz",
  "room_number": "10",
  "contact": "09123456789",
  "monthly_rent": 1800
}
Response: 201 Created
{
  "message": "Tenant created successfully",
  "id": 8
}

PUT Update Tenant

PUT /api/tenants/:id
Example: PUT /api/tenants/4
Headers: Content-Type: application/json
Body:
{
  "name": "JONA PALACIO",
  "room_number": "2",
  "contact": "09999999999",
  "monthly_rent": 1600
}
Response: 200 OK
{
  "message": "Tenant updated successfully"
}

DELETE Tenant

DELETE /api/tenants/:id
Example: DELETE /api/tenants/7
Response: 200 OK
{
  "message": "Tenant deleted successfully"
}

XML Support

You can also send POST/PUT requests using XML. Example:

Headers: Content-Type: application/xml
Body:
<tenant>
  <name>Zoren M</name>
  <room_number>11</room_number>
  <contact>09989874321</contact>
  <monthly_rent>2000</monthly_rent>
</tenant>

Response:
{
  "message": "Tenant created successfully",
  "id": 9
}
